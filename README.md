# Test App

Простий SPA додаток для управління списком елементів з функціоналом авторизації.

## Технології

- React + TypeScript
- Redux Toolkit
- React Router
- Material UI
- Formik + Yup
- Jest + React Testing Library

## Встановлення

```bash
# Клонуйте репозиторій
git clone [url-репозиторію]

# Перейдіть в директорію проекту
cd test-app
# Встановіть залежності
npm install
```

## Запуск

```bash
# Запуск в режимі розробки
npm run dev

# Збірка проекту
npm run build

# Запуск тестів
npm test
```

## Функціонал

- Авторизація користувача (email + пароль)
- Захищені маршрути
- Відображення списку елементів
- Додавання нових елементів
- Редагування існуючих елементів
- Валідація форм
- Адаптивний дизайн
